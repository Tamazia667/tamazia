# Contributing to Tamazia

Merci pour votre intérêt à contribuer ! Quelques règles simples pour faciliter les contributions :

- Ouvrez d'abord une issue pour discuter des changements importants.
- Faites un fork et ouvrez une branche nommée `feature/nom` ou `fix/nom`.
- Créez une PR ciblant `main` (ou la branche de release indiquée).

CI & Releases
- Les actions GitHub automatiques buildent le sous-projet `tamazia-electron` et publient une release lorsqu'un tag `v*` est pushé.
- Ne commitez jamais de secrets (tokens). Utilisez GitHub Secrets pour le CI.

Licence
- En soumettant une PR, vous acceptez que votre contribution soit placée sous la licence MIT du projet.
